# 用户操作手册

> 面向最终用户的「发型宇宙」使用指南。通过本应用，你可以上传照片，让 AI 为你分析脸型、推荐发型、生成换发效果图，并获得专业造型点评。

---

## 界面概览

打开浏览器访问 `http://localhost:3000`，进入「发型宇宙」应用：

- **首页**：展示热门发型、用户作品、核心功能入口
- **底部导航**：首页 / 上传 / 个人中心（移动端布局）

> 本应用采用移动端优先设计，推荐在手机浏览器或微信中打开使用。

---

## 第一章：上传照片

### 1.1 照片要求

为了获得最佳的分析和换发效果，请遵循以下拍摄建议：
- **正面清晰照**：面部正对镜头，不要侧脸
- **光线充足**：避免逆光或过于昏暗的环境
- **无遮挡**：不要戴帽子、墨镜或口罩
- **文件格式**：JPG、PNG、WebP、GIF
- **文件大小**：不超过 10MB

### 1.2 上传步骤

1. 在首页点击 **「看看其他发型的我」** 或底部 **「上传」** 按钮。
2. 选择上传方式：
   - **相机拍照**：点击「相机拍照」卡片，唤起设备相机
   - **相册选择**：点击「从相册选择」按钮，从手机相册挑选

「此处为上传页面截图：相机拍照卡片、相册选择按钮、上传小贴士」

3. 选择照片后进入 **预览页**，确认照片清晰后点击 **「裁剪照片」**。
4. 在裁剪页面，调整裁剪区域使面部居中。可切换比例（1:1 或 3:4）。
5. 点击 **「确认裁剪」**，系统开始上传。

> 上传过程中会自动触发脸型分析。分析完成后自动跳转至发型选择页面。

### 1.3 上传小贴士

页面上提供了三条拍摄建议：
- 请使用正面清晰照片
- 确保光线充足，面部无遮挡
- 避免戴帽子或墨镜

## 第二章：选择发型并生成效果图

### 2.1 浏览发型库

上传成功后进入 **选择发型** 页面：
- 顶部展示你上传的照片缩略图
- 下方是 **发型库**，支持按标签筛选：全部 / 短发 / 中长发 / 长发 / 卷发 / 直发
- 每款发型展示名称、首字头像、标签和简短描述

「此处为发型选择页面截图：照片预览区、风格标签栏、发型卡片网格」

### 2.2 自定义描述

如果你心中有特定想要的发型，可以在输入框中用文字描述：
- 示例：「蓬松微卷的锁骨发，带有法式刘海」
- 输入后点击右侧的 ✨ 按钮，直接以此描述生成

### 2.3 选择发型并生成

1. 点击任意发型卡片选中（选中后卡片会有蓝色边框高亮）。
2. 确认已选发型后，点击底部 **「开始生成」** 按钮。
3. 系统创建异步任务，页面显示生成进度动画与趣味提示语：
   - "AI 造型师正在为你换发型..."
   - "正在分析你的面部轮廓..."
   - "正在匹配最佳发型..."
   - "正在渲染效果图..."

> 生成过程通常需要 10–60 秒，取决于所选模式和当前 LLM 服务负载。

### 2.4 生成模式说明

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| `preview` | 快速预览，生成速度较快 | 快速尝试多款发型 |
| `hd` | 高清模式，画质更精细 | 最终确认、分享 |

默认使用 `hd` 模式。如需切换，需在 API 调用时指定 `mode` 参数（当前前端界面默认 hd）。

### 2.5 免费次数

每日默认提供 **3 次免费生成机会**。页面底部显示今日剩余次数：
- 次数用尽后，可通过 **分享应用给好友** 解锁额外次数
- 分享成功后自动增加 1 次

### 3.1 效果图对比

生成完成后自动跳转至 **生成结果** 页面：
- **对比滑块**：左右拖动滑块，对比换发前后的效果
- 滑块左侧为原图，右侧为 AI 生成的换发效果图

「此处为结果页面截图：对比滑块、风格标签、AI 点评卡片」

### 3.2 AI 造型师点评

页面下方展示 AI 造型师的专业点评，包含：

#### 人格风格切换
点击顶部的三个按钮切换点评风格：
- **温柔闺蜜**（默认）：亲切温暖，像好朋友一样给出建议，多用 emoji
- **毒舌造型师**：犀利直白，吐槽中带关心，幽默风趣
- **知识型博主**：专业严谨，引用流行趋势和发质科学知识

#### 六维雷达图
展示六个维度的评分（满分 100）：
- 脸型适配度
- 发质匹配度
- 风格气质匹配度
- 情绪价值
- 专业冷知识
- 幽默互动

#### 点评正文
按类别分段的点评内容，如「造型点评」「脸型匹配」「专业建议」。

#### 造型建议
AI 给出的实用小贴士，例如：
- "搭配暖棕色会更显白"
- "定期做发膜护理"
- "尝试侧分刘海"

### 3.3 重新生成或分享

页面底部提供三个操作：
- **换一个发型**：返回发型选择页，尝试其他发型
- **分享长图**：跳转至分享页面，生成精美分享卡片
- **重新上传照片**：返回上传页，换一张照片重新开始

## 第四章：分享长图

### 4.1 生成分享卡片

在结果页点击 **「分享长图」**，进入分享页面：
- 页面展示自动生成的分享卡片预览
- 卡片包含：效果图、发型名称、AI 点评摘要、造型建议、应用品牌信息

「此处为分享页面截图：分享卡片预览、保存/复制/分享按钮」

### 4.2 保存与分享

底部提供三个操作按钮：
- **保存图片**：将分享卡片保存为 PNG 图片到本地相册
- **复制链接**：复制当前页面链接到剪贴板

## 常见问题 FAQ

### Q1：上传照片时提示「请上传图片文件」？

确保选择的文件是图片格式（JPG、PNG、WebP、GIF）。如果确认是图片文件但仍报错，尝试：
- 换一张照片重新上传
- 检查照片是否损坏（能否正常在相册中打开）

### Q2：照片上传后脸型分析失败？

脸型分析依赖 LLM Vision 服务。如果分析失败：
1. 检查网络连接是否正常
2. 确认后端 `.env` 中已配置有效的 `LLM_API_KEY`
3. 如使用 Mock Provider，会返回默认的椭圆脸数据，不影响后续功能体验
4. 分析失败不会阻塞流程，仍可继续选择发型和生成效果图

### Q3：生成效果图时一直显示「AI 生成中...」？

正常情况下生成需要 10–60 秒。如果长时间无响应：
1. 检查网络连接
2. 查看后端日志确认任务状态：`docker-compose logs -f backend`（如使用 Docker）
3. 如使用 fal.ai 等外部服务，可能因队列繁忙导致延迟
4. 前端会自动轮询任务状态，无需手动刷新页面

### Q4：生成结果不满意，如何重新生成？

- 在结果页点击 **「换一个发型」**，选择其他发型重新生成
- 或在生成页使用 **自定义描述** 精确表达你想要的效果
- 每日免费次数有限，分享应用可解锁额外次数

### Q5：发型库中没有我想要的发型？

可以使用 **自定义描述** 功能，在输入框中用文字描述你想要的发型：
- 描述越具体，生成效果越接近预期
- 建议使用英文描述（如 "short bob with bangs"），图像生成模型对英文理解更准确

### Q6：分享卡片保存失败？

分享卡片使用 `html-to-image` 库在前端生成。如果保存失败：
1. 确保浏览器允许下载文件
2. 尝试使用「复制链接」功能手动分享
3. 部分浏览器（如微信内置浏览器）可能限制自动下载，建议在系统浏览器中打开

### Q7：如何切换 LLM 服务提供商？

这是开发者/管理员操作。在 `.env` 中修改 `LLM_PROVIDER`：
- `mock` — 无 API Key，返回模拟数据（开发测试用）
- `openai` — OpenAI GPT-4o + DALL-E-3
- `falai` — fal.ai SDXL + InstantID（推荐用于换发）
- `aliyun` — 阿里云通义万相 + 通义千问
- `volcengine` — 火山引擎豆包
- `dashscope` — 阿里云 DashScope（Qwen2-VL）

详见 [DEVELOPER.md](./DEVELOPER.md) 中的 LLM Provider 配置说明。

### Q8：数据存储在哪里？

默认使用 **SQLite** 本地数据库（`./data/hair_multica.db`），图片存储在 `./uploads/` 目录下。
- 生产环境建议切换至 PostgreSQL 和对象存储（S3）
- 配置方式详见 [DEVELOPER.md](./DEVELOPER.md)
